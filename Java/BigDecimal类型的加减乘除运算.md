### Java中BigDecimal类型的加减乘除运算

```java
add():加法; subtract():减法; multiply():乘法; divide():除法;
```

### compareTo比较大小

- a小于b：a.compareTo(b) == -1

- a等于b：a.compareTo(b) == 0

- a大于b：a.compareTo(b) == 1

- a大于等于b：a.compareTo(b) > -1

- a小于等于b：a.compareTo(b) < 1
